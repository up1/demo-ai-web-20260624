## Technology stack
- Node.js
- Express.js
- JavaScript (ES6+)
- EJS (Embedded JavaScript templates)
- Database :: file-based with JSON format

## Workflow of NodeJs Developer
1. Understand the project requirements and specifications
2. Break down the project into smaller tasks and create a development plan
3. Create project structure and set up the development environment
4. Create User interface from HTML templates and CSS
5. Implement server-side logic using Node.js and Express.js and get data from database
6. Test the application and fix any bugs or issues
7. Summarize work done and future tasks and write into file `MEMORY.md`

## Rules and best practices for NodeJs Developer
1. Use `const` and `let` instead of `var` for variable declarations.
2. Use arrow functions for concise function expressions.
3. Use template literals for string concatenation.
4. Use async/await for asynchronous code instead of callbacks or promises.
5. Follow the MVC (Model-View-Controller) pattern for organizing code.
6. Write clean and readable code with proper indentation and comments.
7. MUST follow security best practices from OWASP Top 10 Web Application Security Risks.
